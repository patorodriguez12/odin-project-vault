
### ¿Qué es Odin Project?

Odin Project es un curso sobre desarrollo web de código abierto cuyo objetivo es brindar a sus estudiantes las bases y las herramientas para aprender a programar, entendiendo todo desde lo mas primitivo proporcionando la capacidad de resolución de problemas.

### ¿Deberías saltarte alguna unidad o cursarlo de manera no lineal?

Por supuesto que no, este curso esta pensado para cursarse de manera creciente y progresiva, si te saltas contenido vas a estar perdiendo conocimiento útil, por lo que es mejor tomárselo con calma y leer bien todo para entender, no se busca aprender una tecnología o resolver algo concreto, se busca desarrollar la mentalidad de programador y resolver problemas.
